# Portfolio Optimizer

**Markowitz Efficient Frontier × Monte Carlo Simulation**

不確実性下での最適資産配分を、数理最適化と確率的シミュレーションの対比で可視化するツール。

---

## 概要

現代ポートフォリオ理論（Markowitz, 1952）に基づき、複数資産の最適ウェイトを導出する。

- **モンテカルロシミュレーション**：ランダムウェイトを大量サンプリングし、リスク×リターン空間に投影
- **効率的フロンティア**：`scipy.optimize.minimize` による解析的最適化曲線をオーバーレイ
- **最大シャープレシオ最適化**：リスク調整後リターンを最大化する配分を導出

2つのアプローチ（確率的探索 vs 数理最適化）を一画面で対比させることで、最適解の意味を直感的に理解できる設計にした。

## 設計上の工夫

### 1. モンテカルロ × 解析解の対比
モンテカルロの散布図（確率的）と効率的フロンティア曲線（解析的）を重ねて描画。
「なぜ解析解が散布図の包絡線になるのか」が視覚的に自明になる構造。

### 2. ディリクレ分布によるウェイトサンプリング
`np.random.dirichlet(np.ones(n))` を使用。
単純な一様サンプリングと比較して、ウェイト空間の均一なカバレッジが得られる。

### 3. キャッシュ設計
`@st.cache_data(ttl=3600)` でAPI取得結果をキャッシュ。
パラメータ変更時の再計算コストを最小化。

### 4. 銘柄の自動バリデーション
取得失敗銘柄を自動除外し、有効銘柄のみで計算を継続。

## 背景・動機

東京理科大学で金融工学（ポートフォリオ理論・最適化・確率過程）を専攻。
理論で学んだマルコウィッツモデルを「実際に動いて触れるもの」として実装した。

意思決定理論の観点から、このツールは「不確実性下での合理的選択」の構造を可視化するものとして設計している。
単なるポートフォリオ計算にとどまらず、リスク×リターンのトレードオフ空間そのものを探索できる点に価値がある。
